# CareerAI

CareerAI is a full-stack career platform with a Flask backend and a React frontend.

## Backend

Path: `backend/`

Run:

```bash
cd backend
source ../.venv/bin/activate
python app.py
```

Backend default URL: `http://127.0.0.1:5001`

## Frontend

Frontend source is available in this repository via the merged `master` history.

Run:

```bash
npm install
npm start
```

Frontend default URL: `http://localhost:3000`
